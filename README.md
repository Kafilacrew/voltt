# Voltt – One-page site (React + Vite + Tailwind)

Landing page built from the Voltt design in `pencil-new.pen`.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (e.g. http://localhost:5173).

## Build

```bash
npm run build
npm run preview   # preview production build
```

## Assets

Add images to **`public/assets/`**. The app expects these paths (see `public/assets/README.md` for the full list):

- `hero-bg.jpg` – hero background
- `product-1.jpg` … `product-4.jpg` – product cards
- `quality.jpg`, `value.jpg`, `taste.jpg`, `trust.jpg` – feature cards
- `bundles.jpg` – Stock Up section
- `goal-muscle.jpg`, `goal-energy.jpg`, `goal-clean.jpg` – goal cards

Missing images are hidden so the layout still works.

## Stack

- **React 18** (JSX)
- **Vite 6**
- **Tailwind CSS 3** (Montserrat + Inter, design tokens for brand red/yellow/brown/teal)
