# Assets folder

Add your images here. They will be served at `/assets/...`.

## Suggested filenames (used by the app)

- **hero-bg.jpg** – Hero section background (full-width, tall; e.g. 1920×1080)
- **product-1.jpg**, **product-2.jpg**, **product-3.jpg**, **product-4.jpg** – Product cards in “Choose Your Power”
- **quality.jpg**, **value.jpg**, **taste.jpg**, **trust.jpg** – “Premium Nutrition” feature cards
- **bundles.jpg** – “Stock Up. Save Big.” section
- **goal-muscle.jpg**, **goal-energy.jpg**, **goal-clean.jpg** – “Your Goal. Our Fuel.” cards

If a file is missing, the UI will hide the image or show a placeholder so the layout still works.
